Sobre mim

Olá, sou Alexandre Lages, desenvolvedor Java Pleno com uma trajetória diversificada no mundo da programação. Já mergulhei em várias tecnologias, destacando-me no desenvolvimento backend com Java (Spring), além de explorar o universo frontend com Angular, React, Vue.js, HTML e CSS. Minha versatilidade se estende a diferentes plataformas, desde a programação para desktop utilizando VB.NET (.NET), até o desenvolvimento mobile com Android, Ionic e Swift. Além disso, desenvolvo web com Python e o framework Django. Sou formado em Ciência da Computação pela Universidade Federal do Piauí (UFPI) e conquistei meu mestrado na mesma instituição. Proporcionando uma visão abrangente da Ciência da Computação. Com a ambição de aprimorar minhas habilidades técnicas e alcançar o status de desenvolvedor sênior a curto prazo, busco constantemente desafios que impulsionem meu crescimento profissional. Acredito profundamente na colaboração e trago para cada projeto não apenas competência técnica, mas também a habilidade de trabalhar harmoniosamente em equipe.
